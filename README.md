# Calculadora Java

Calculadora desenvolvida em Java com suporte a operações básicas: adição, subtração, multiplicação e divisão.

## Funcionalidades

- Operações: +, -, *, /
- Validação de divisão por zero
- Tratamento de operações inválidas
- Loop contínuo até o usuário encerrar

## Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Tratamento de exceções (throw, try/catch)
- Separação de responsabilidades entre classes

## Como executar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe Main
